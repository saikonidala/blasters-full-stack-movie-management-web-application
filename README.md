# 🎬 Full Stack Movie Management Web Application

## 📌 Project Overview

This is a full-stack web application that allows users to manage movies efficiently. Users can add, update, delete, and search movies along with uploading movie posters.

---

## 🚀 Features

* 🔐 User Registration & Login
* ➕ Add new movies with posters
* ✏️ Update movie details
* ❌ Delete movies
* 🔍 Search movies by name
* 🎭 Filter movies based on genre

---

## 🛠️ Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* MYSQl

## ⚙️ Installation & Setup

### 1. Clone the repository

```
https://github.com/saikonidala/blasters-full-stack-movie-management-web-application
```

### 2. Navigate to project folder

```
cd blasters-full-stack-movie-management-web-application
```

### 3. Install dependencies

```
npm install
```

### 4. Start the backend server

```
node backend/server.js
```

### 5. Run frontend

Open `frontend/index.html` in your browser
